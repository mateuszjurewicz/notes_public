# Tmux

https://gist.github.com/MohamedAlaa/2961058

- `ctrl + b + n` - move to next window 
- `ctrl + b + p` - move to previous window
- `ctrl + b + %` - split window vertically
- `ctrl + b + "` - split window horizontally
